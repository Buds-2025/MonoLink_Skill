# 提示词模板

## 基础模板

生成一张黑白诗性拼贴摄影图像，采用【版式】。整体像实验摄影书内页，外部保留宽白边，画面之间用白色留缝分隔。主图表现【符合现实逻辑的主体与场景】，主图必须保持真实可信的身体结构、空间关系、重力方向和接触关系。副图表现【用于解释或拓展主图局部的内容】，副图可以更超现实，但必须从主图中的【合理接口】发展出来。两个画面之间通过【组合/延伸/相似/对照/递进】关系建立联系，关系蓝图是【主图接口】 -> 【边界连接点】 -> 【副图承接形态】 -> 【视觉证据】。如果是延伸，写清起点、方向、连续媒介和落点；如果是相似，写清至少两项视觉证据，例如轮廓、走势、结构、密度节奏或尺度反差。具体表现为【关系描述】。避免【最可能失败条件】。整体使用黑白灰色调，灰白纸张背景，明显胶片颗粒，轻微灰尘和划痕，轻微失焦，旧照片翻拍质感。构图克制，主体偏小，大面积留白，气氛安静、孤独、冷峻、梦境化。no text，no color，no clutter，no glossy commercial photography，no weak visual relation，no random juxtaposition。

## 关系强化模板

生成一张黑白诗性拼贴摄影图像，采用【版式】。这张图的主图是【主图内容】，必须符合现实世界的身体结构、空间关系、重力方向和接触逻辑。副图是【副图内容】，用于【类比/延伸/相似/对照/递进/超现实阐释】主图中的【具体局部】。主关系是【组合/延伸/相似/对照/递进】，并且要让这种关系一眼可见，不能只是概念上有关。

关系蓝图：【主图接口】 -> 【边界连接点】 -> 【副图承接形态】 -> 【视觉证据】 -> 【失败条件】。

如果主关系是延伸，必须写清：起点是【主图端点/出口/痕迹末端】，方向是【透视轴线/运动方向/重力方向/视线方向】，连续媒介是【副图中承接的线、裂纹、光带、烟、水流、花瓣、鸟群、纹理】，落点是【副图中从同一轴线继续展开的位置】。

如果主关系是相似，必须写清至少两项视觉证据：【轮廓相似/走势相似/结构相似/密度节奏相似/尺度反差】。两幅图必须通过【关系具体描述】建立强联系，尤其要让【合理接口/边界处/路径交接/轮廓相似/尺度反差】清楚成立。副图必须从主图的合理接口发展出来，不能凭空出现。观者第一眼就应理解两图为何拼在一起，并能感到副图解释了主图的来源、结果、反面、隐喻或潜在结构。之后再加入实验摄影书内页感，外部保留宽白边，画面之间用白色留缝分隔，使用黑白灰色调，灰白纸张背景，明显胶片颗粒，轻微灰尘和划痕，轻微失焦，构图克制，主体偏小或连接点突出，大面积留白，情绪安静、孤独、冷峻、超现实。no text，no color，no clutter，no glossy commercial photography，no weak visual relation，no random juxtaposition。

## 主副图规划模板

在写最终提示词前，先输出或内化以下五项：

1. 主图：【现实中成立的画面】
2. 副图：【解释、放大、反转或超现实化主图哪个局部】
3. 接口：【主图中的合理连接点】
4. 主关系：【组合/延伸/相似/对照/递进】
5. 意义句：【副图让主图多出了什么意义】
6. 边界连接点：【接口如何靠近或指向拼接边界】
7. 视觉证据：【延伸四要素或相似至少两项证据】
8. 失败条件：【最需要禁止的弱关系退化】

## 填写方法

- 【版式】：左右双联画 / 上下双联画 / 三联画 / 四宫格
- 【主图内容】：例如“夜色中的城市高架车流，灯迹形成分叉与汇聚的路径网络”
- 【副图内容】：例如“微距视角下大群蚂蚁沿地面形成类似分叉网络的行进路径”
- 【合理接口】：例如“道路流线的末端、裙摆边缘、手中物、影子边缘、烟雾出口、裂缝末端”
- 【关系具体描述】：例如“左右两图都突出分叉、汇聚、流动的路径结构，让城市车流与蚂蚁军团路线呈现一眼可见的形态相似，同时形成宏观与微观的强对照”
- 【视觉证据】：延伸写“起点、方向、连续媒介、落点”；相似写“轮廓、走势、结构、密度节奏、尺度反差”中的至少两项。
- 【最可能失败条件】：例如“禁止两图只是普通并列；禁止连接点远离边界；禁止只有概念相似而无形状证据”

## 关系句先行法

生成前，先写一句关系句：

- 火焰就是人的头。
- 车流像蚂蚁军团的路径网络。
- 车辙继续变成行人的脚印。
- 树枝像张开的手掌。
- 裙摆边缘的碎花落到地面，变成真实花丛。
- 手中握着的一小束花，在副图中被放大成正在绽开的花田。

再把这句话压进提示词，图像会更稳定。
