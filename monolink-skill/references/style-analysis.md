# 风格分析

## 整体属性

这类作品可以概括为：黑白诗性拼贴摄影 + 实验摄影书版式 + 象征性意象组合。

它的价值不只是“好看”，而是通过图像之间的结构关系，让普通题材产生隐喻和心理张力。

## 颜色特点

1. 单色系统。只保留黑、白、灰，不进入彩色叙事。
2. 白不是纯白，而是偏灰纸白、雾白、旧纸白。
3. 黑不是广告级纯黑，而是颗粒化、边缘略晕的墨黑。
4. 灰阶丰富，是情绪的主要承载层。

## 画面质感

1. 胶片颗粒明显。
2. 常见灰尘点、细小划痕、轻度翻拍感。
3. 看上去像摄影书页面、冲印照片、旧杂志图版，而非现代高清商业大片。
4. 允许轻微模糊，使画面更像记忆而非现场说明。

## 主体语言

主体常常是“轮廓优先”的对象：

- 飞鸟
- 行人
- 树枝
- 路灯
- 烟雾
- 羽毛
- 雨伞
- 汽车
- 海面、沙丘、道路痕迹

这些元素的共同点是：易形成剪影，可以被缩小后仍有辨识度，适合和其他对象建立轮廓呼应。

## 情绪气质

保持静、冷、远、空、孤、克制，带轻微超现实意味。

它不是讲清楚故事，而是给出可被感受的关系。
