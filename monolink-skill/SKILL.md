# monolink-skill

## 第一原则

现实逻辑优先，关系第二，风格第三。

这类图像的成败，不取决于它是不是黑白、有无颗粒、有没有白边，而取决于两张或多张图之间是否真的发生了明确、强烈、可一眼识别的关系。

如果只有黑白灰、颗粒、留白，但图像之间没有强关系，那只能算“像这种风格”，不能算“生成成功”。

但“强关系”不能靠生搬硬套获得。主图必须先符合现实世界的客观规律、身体结构、空间关系、重力方向、接触关系和动作因果。副图可以更超现实，但必须从主图中一个合理的视觉接口、动作接口或意义接口发展出来。

## 主图与副图

每次生成时，先建立主副图分工：

- `主图`：现实锚点，是被类比、拓展、延伸、比较或反转的对象。主图必须符合客观规律，不能为了拼贴效果破坏现实逻辑。
- `副图`：阐释图，是用来类比、延伸、比较、相似、对照或超现实化主图某个局部的图。副图可以超现实，但不能无来源地凭空出现。
- `接口`：主图中允许关系发生的位置。接口必须符合物理或叙事逻辑，例如手中所拿之物、裙摆与地面接触处、影子、道路尽头、烟囱出口、窗外视线、身体动作方向、物体裂缝、痕迹末端。

不要先问“怎样拼得像”，要先问：“主图里哪个真实存在的局部，允许副图合理地接上、放大、类比或反转？”

## 使用流程

1. 先确定主图：把用户主题转成一个现实中成立的画面。
   - 人物的身体、姿态、衣物、手部、视线、影子和地面关系要合理。
   - 物体的支撑、重力、接触点、光影和空间方向要合理。
   - 场景不能为了关系效果凭空生成不可能的连接。
2. 再确定副图功能：副图用来类比、延伸、比较、相似、对照、递进或超现实阐释主图的哪个局部。
3. 找到接口：选择主图中一个物理或叙事上允许关系发生的位置。
   - 允许：裙摆边缘延伸到地面花丛；手里拿的花在副图中绽开；烟从烟囱口进入云层；路尽头延续为河流；影子变成另一种形态。
   - 避免：花枝从头边凭空生出；火焰接在没有截断逻辑的身体旁；道路随机接到无方向关系的纹理；两个相似物只因“长得像”被硬并列。
4. 把用户主题提炼成一句关系句：
   - 火焰就是人的头。
   - 车流像蚂蚁军团的行进路径。
   - 掌纹像海浪退去后的岸线。
   - 树枝像人的手掌与神经。
5. 只选择一个最强主关系：
   - `组合`：两个画面在边界处拼合后，成为一个新主体或新意象。
   - `延伸`：一个画面中的线条、轨迹、势能、流动方向，在另一个画面中继续发展。
   - `相似`：两个不同对象在轮廓、结构、走势、质地或节奏上高度相似。
   - `对照`：两个画面在尺度、轻重、远近、动静、自然与人工、微观与宏观等维度上形成显著反差。
   - `递进`：两个画面构成前后次序、时间推进、动作结果或心理推进。
6. 决定版式：
   - 左右双联画：适合并列比较、相似、对照。
   - 上下双联画：适合组合、递进、路径延伸。
   - 三联或四宫格：适合多重递进，但前提是每格关系仍然清楚。
7. 把关系写成硬性视觉要求。不要只写“有联系”，要写清主图、副图、接口、物理依据、路径延续、轮廓相似或尺度对照。
8. 写出关系蓝图：`主图接口 -> 边界连接点 -> 副图承接形态 -> 视觉证据 -> 失败条件`。没有蓝图就不要进入风格描述。
9. 最后再加入黑白灰、胶片颗粒、白边、轻微划痕、强留白、摄影书内页质感和 `no text` 等风格壳层。
10. 输出前按失败判定标准检查。

## 五种主关系的执行标准

### 组合

两个画面在边界处拼合后，成为一个新主体或新意象。边界位置必须正好对应身体、物体或轮廓的连接点，并且主图中要有现实理由允许该位置被替换、遮挡、截断或承接。两部分不能只是上下相邻，必须让观者明确感觉“它们原本就是一个整体”。如果观者还会觉得“这只是上面一张火焰图，下面一张人图”，说明失败。

### 延伸

一个画面中的线条、轨迹、势能、流动方向，在另一个画面中继续发展。延伸不能只靠“题材相关”，必须同时具备四个可见条件：

1. 起点：主图中有明确端点或出口，例如裙边、路尽头、烟雾出口、手中物、影子边缘、裂缝末端或痕迹尽头。
2. 方向：主图的运动方向、透视轴线、重力方向或视线方向清楚指向拼接边界。
3. 连续媒介：副图中出现能承接该方向的线、光带、裂纹、烟、花瓣、水流、脚印、鸟群、纹理或阴影。
4. 落点：副图的承接形态从同一轴线或同一边界区域继续展开，而不是随机出现在另一处。

提示词必须写清“从哪里开始、朝哪里走、通过什么继续、在副图哪里落下”。若延伸点离边界太远、方向轴线断开、上图线索和下图线索不共线，优先判定失败。

### 相似

两个不同对象在轮廓、结构、走势、质地或节奏上高度相似。相似性必须是视觉上明显的，而不只是概念上相关。使用相似关系时，至少满足以下五项中的两项，且必须在提示词中点名：

1. 轮廓相似：外形、负形、剪影或边缘曲线明显对应。
2. 走势相似：主要方向、弧线、分叉、汇聚、旋转或起伏节奏一致。
3. 结构相似：骨架、网状结构、枝杈、路径系统、层级关系或重复单元相似。
4. 密度节奏相似：疏密、间距、群聚方式、重复频率形成一眼可见的呼应。
5. 尺度反差明确：宏观与微观、远景与近景、自然与人工之间形成强对照，同时仍共享上述视觉结构。

相似不能只停留在“长得像”，还要说明副图为何能解释主图，例如把衣纹解释为地形、把掌纹解释为海岸、把车流解释为生物群体路径。若相似只能靠文字解释，画面中没有轮廓、走势、结构或节奏证据，优先判定失败。

## 强关系蓝图

生成提示词前，先内化或输出以下五项。它们必须进入最终提示词的关系描述，而不能只停留在分析里：

1. 主图接口：主图中真实存在、允许关系发生的具体位置。
2. 边界连接点：该接口如何靠近拼接边界，或如何用方向、视线、轨迹指向边界。
3. 副图承接形态：副图用什么具体形态接住主图，例如裂纹、花丛、云痕、鸟群、路径、影子、浪线。
4. 视觉证据：观者不用解释也能看见的相同轴线、同形轮廓、连续路径、疏密节奏或尺度反差。
5. 失败条件：如果模型把两图做成普通并列，最可能弱在哪里，并在提示词中明确禁止。

最终提示词要把关系写成可拍摄、可构图的要求，例如“车道线必须贴近下方边界，并与下图裂纹保持同一透视轴线”，不要只写“道路和裂纹有延伸感”。

### 对照

两个画面在尺度、轻重、远近、动静、自然与人工、微观与宏观等维度上形成显著反差。对照不能只是题材不同，必须有结构性反差；同时仍要保留某种联系，否则会变成无关并列。优先让主图保持现实，副图负责放大其隐含的反面、代价、来源或结果。

### 递进

两个画面构成前后次序、时间推进、动作结果或心理推进。两图之间要能说出明确的前后关系句子，最好由路径、痕迹、手势、视线、风向或动作方向支撑。若两图顺序互换也成立，说明递进不够强。

## 硬性规则

### 先定关系，再定主体

生成前先回答：“主图是什么？副图解释什么？接口在哪里？这次最重要的关系是什么？”

只有回答清楚，才允许进入主体与版式设计。

### 边界必须有意义

拼接边界不是装饰分割线，而是关系发生区。关键连接点优先靠近边界，主体联系尽量在边界处完成。但边界不能制造不符合现实逻辑的凭空生长、凭空断裂或错误接触。如果边界上下或左右没有任何强连接，或连接点在现实中没有来源，优先判定为失败。

### 关系必须一眼可感知

图像关系要视觉上明显，不接受“解释之后才懂”的弱关系。写提示词时，必须明确使用以下表达中的一种或多种：

- 让这种联系一眼可见
- 让延伸关系非常明显
- 让相似轮廓具有强识别度
- 让边界处完成组合
- 让对比与联系同时清晰成立
- 让观者第一眼就看懂两图为何拼在一起
- 主图保持现实可信，副图从主图的合理接口中发展出来
- 让关系既符合物理逻辑，又具有隐喻张力

### 主体宁少勿多，但关系必须更强

可以少元素，但不能少关系。这类图像不是靠堆元素取胜，而是靠高强度连接取胜。

### 风格词只能服务关系，不得盖过关系

黑白灰、颗粒、白边、胶片感、留白只是辅助层。若模型偏向生成“漂亮的黑白拼贴”而非“有明确关系的拼贴”，必须收紧提示词，把关系描述写得更具体。

## 提示词模板

生成一张黑白诗性拼贴摄影图像，采用【版式】。先确保主图是一个符合现实逻辑的画面：【主图内容】。副图用于【类比/延伸/相似/对照/递进/超现实阐释】主图中的【具体局部】，副图内容为【副图内容】。两幅图之间的主关系是【组合/延伸/相似/对照/递进】，并让这种关系一眼可见。关系必须从主图中的【合理接口】发生，接口要靠近拼接边界或明确指向拼接边界。关系蓝图为：【主图接口】 -> 【边界连接点】 -> 【副图承接形态】 -> 【视觉证据】。如果是延伸，必须写清起点、方向、连续媒介和落点；如果是相似，必须写清至少两项视觉证据，例如轮廓、走势、结构、密度节奏或尺度反差。具体表现为【关系具体描述】，同时符合物理方向、接触关系、身体结构或叙事因果，不能只是形式上的拼贴。避免【最可能失败条件】。整体保持实验摄影书内页感，外部保留宽白边，画面之间用白色留缝分隔，使用黑白灰色调，灰白纸张背景，明显胶片颗粒，轻微灰尘划痕，轻微失焦，构图克制，主体偏小或连接点突出，大面积留白，气氛安静、孤独、冷峻、超现实。no text，no color，no clutter，no glossy commercial photography，no weak visual relation，no random juxtaposition。

## 失败判定标准

出现以下情况，直接判为失败，需重写提示词或重生成：

1. 两张图拆开看都成立，拼在一起却没有新增意义。
2. 观者第一眼看不出两图为什么被放在一起。
3. 边界处没有承担关系任务，只是普通分隔。
4. 关系只能靠解释成立，不能靠视觉成立。
5. 黑白风格出来了，但联系过弱。
6. 延伸不连续，相似不明显，对照不强，组合不完整，递进不清楚。
7. 主图为了关系效果破坏了现实逻辑、身体结构、重力方向或空间接触关系。
8. 副图没有从主图的合理接口发展出来，而是凭空出现。
9. 只做了视觉相似，没有说明副图对主图的意义、来源、结果、反面或隐喻。
10. 延伸关系缺少起点、方向、连续媒介或落点中的任一项。
11. 延伸线索没有靠近拼接边界，或主图轴线与副图承接轴线断开。
12. 相似关系没有至少两项可见证据，例如只有概念相似，没有轮廓、走势、结构、密度节奏或尺度反差。
13. 两张图分开看不影响意义，拼接没有制造新的视觉判断。

## 输出策略

如果用户只给一个主题，例如“快要下雨的天空”“城市的车流和蚂蚁军团路径”“火焰头的人”，按以下顺序执行：

1. 先写一句关系判断。
2. 明确现实主图。
3. 明确副图功能。
4. 明确主关系。
5. 明确版式。
6. 明确物理接口、连接点或对照点。
7. 再写完整提示词或直接出图。
